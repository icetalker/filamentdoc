## 概述

卡片(Card)组件可用于在卡片内渲染信息列表组件：

```php
use Filament\Infolists\Components\Card;

Card::make()
    ->schema([
        // ...
    ])
```

## 在卡片内使用网格列宽

你可以使用 `columns()` 方法轻松地在卡片中创建 [Grid](grid):

```php
use Filament\Infolists\Components\Card;

Card::make()
    ->schema([
        // ...
    ])
    ->columns(2)
```
