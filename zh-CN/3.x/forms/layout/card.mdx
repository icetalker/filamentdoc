## 概述

卡片(Card)组件可用于在卡片内渲染表单组件：

```php
use Filament\Forms\Components\Card;

Card::make()
    ->schema([
        // ...
    ])
```

## 在卡片内使用网格列

使用 `columns()` 方法，可以快速在卡片内创建[网格](grid)：

```php
use Filament\Forms\Components\Card;

Card::make()
    ->schema([
        // ...
    ])
    ->columns(2)
```
