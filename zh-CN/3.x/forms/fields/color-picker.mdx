## 概述

拾色器(ColorPicker)组件允许你以一系列格式选择颜色。

默认情况下，组件使用 16 进制格式：

```php
use Filament\Forms\Components\ColorPicker;

ColorPicker::make('color')
```

## 设置颜色格式

虽然默认为 16 进制格式，你也可以使用其他格式：

```php
use Filament\Forms\Components\ColorPicker;

ColorPicker::make('hsl_color')
    ->hsl()

ColorPicker::make('rgb_color')
    ->rgb()

ColorPicker::make('rgba_color')
    ->rgba()
```
