## 概述

复选框列字段(CheckboxColumn)运行你在表格中渲染复选框，可用于在不打开新页面或模态框的情况下更新数据库记录：

```php
use Filament\Tables\Columns\CheckboxColumn;

CheckboxColumn::make('is_admin')
```
