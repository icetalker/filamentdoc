颜色字段让你可以通过 CSS 颜色定义预览颜色，一般使用一种支持的格式(HEX、HSL、RGB、RGBA)通过拾色器字段进行输入。


```php
use Filament\Tables\Columns\ColorColumn

ColorColumn::make('color')
```

## 允许将颜色复制到剪贴板

你可以将其设置成可复制。这样的话，点击预览时，便会将 CSS 值复制到剪贴板中，并可指定自定义确认消息和持续时长(ms)。该特性只有在 SSL 启用时有效。

```php
use Filament\Tables\Columns\ColorColumn

ColorColumn::make('color')
    ->copyable()
    ->copyMessage('Color code copied')
    ->copyMessageDuration(1500)
```

### 自定义复制到剪贴板的文本

使用 `copyableState()` 方法，你可以自定义复制到剪贴板的文本：

```php
use Filament\Tables\Columns\ColorColumn;

ColorColumn::make('color')
    ->copyable()
    ->copyableState(fn (string $state): string => "Color: {$state}")
```

本函数中，你可以使用 `$record` 函数访问整条记录：

```php
use App\Models\Post;
use Filament\Tables\Columns\ColorColumn;

ColorColumn::make('color')
    ->copyable()
    ->copyableState(fn (Post $record): string => "Color: {$record->color}")
```
