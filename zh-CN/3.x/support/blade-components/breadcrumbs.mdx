## 概述

面包屑组件用作渲染简单、线性导航，告知用户他们在应用内的当前位置：

```blade
<x-filament::breadcrumbs :breadcrumbs="[
    '/' => 'Home',
    '/dashboard' => 'Dashboard',
    '/dashboard/users' => 'Users',
    '/dashboard/users/create' => 'Create User',
]" />
```

数组的键是用户可以点击导航的网址，数组的值是每个链接显示的文本。
