## 概述

文本区(Textarea) 允许你使用多行字符串：

```php
use Filament\Forms\Components\Textarea;

Textarea::make('description')
```

## 调整文本区大小

你可以定义 `rows()` 及 `cols()` 方法，来修改文本去的大小：

```php
use Filament\Forms\Components\Textarea;

Textarea::make('description')
    ->rows(10)
    ->cols(20)
```

### 文本去自动大小

通过设置 `autosize()` 方法，你也可以让文本区根据其内容自动调整大小：

```php
use Filament\Forms\Components\Textarea;

Textarea::make('description')
    ->autosize()
```

## 文本区验证

除了[验证](../validation)页面上的所有规则，文本区也有一些专门的验证规则。

### 长度验证

使用 `minLength()` 和 `maxLength()` 方法，你可以限制输入的长度。这些方法同时会对前端和后端进行验证：

```php
use Filament\Forms\Components\Textarea;

Textarea::make('description')
    ->minLength(2)
    ->maxLength(1024)
```

使用 `length()`，你可以指定输入框的准确长度。该方法同时添加前端和后端验证：

```php
use Filament\Forms\Components\Textarea;

Textarea::make('question')
    ->length(100)
```
