标签列(TagsColumn)将一个数组渲染成标签列表：

```php
use Filament\Tables\Columns\TagsColumn;

TagsColumn::make('tags')
```

请确保将 `array` [casts](https://laravel.com/docs/eloquent-mutators#array-and-json-casting) 添加到对应的模型属性中：

```php
use Illuminate\Database\Eloquent\Model;

class Post extends Model
{
    protected $casts = [
        'tags' => 'array',
    ];

    // ...
}
```

## 使用分隔符

除了使用数组，你也可以使用字符串分隔，请将分隔符传入到 `separator()` 中： 

```php
use Filament\Tables\Columns\TagsColumn;

TagsColumn::make('tags')->separator(',')
```
