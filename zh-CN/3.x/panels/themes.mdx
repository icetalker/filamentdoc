## 修改颜色

在[配置](configuration)中，你可以轻松地修改使用的颜色。Filament 随带了 6 个预定义颜色，你可以在框架的任意地方使用。他们可以自定义如下：

```php
use Filament\Panel;
use Filament\Support\Colors\Color;

public function panel(Panel $panel): Panel
{
    return $panel
        // ...
        ->colors([
            'danger' => Color::Rose,
            'gray' => Color::Gray,
            'info' => Color::Blue,
            'primary' => Color::Indigo,
            'success' => Color::Emerald,
            'warning' => Color::Orange,
        ]);
}
```

`Filament\Support\Colors\Color` 类包含了所有 [Tailwind CSS 调色板](https://tailwindcss.com/docs/customizing-colors)中的所有颜色选项：

此外，你也可以用一个 RGB 值数组的格式，传入你自己的颜色：

```php
$panel
    ->colors([
        'primary' => [
            50 => '238, 242, 255',
            100 => '224, 231, 255',
            200 => '199, 210, 254',
            300 => '165, 180, 252',
            400 => '129, 140, 248',
            500 => '99, 102, 241',
            600 => '79, 70, 229',
            700 => '67, 56, 202',
            800 => '55, 48, 163',
            900 => '49, 46, 129',
            950 => '30, 27, 75',
        ],
    ])
```

### 生成调色板

如果你想基于十六进制值或者 RGB 值生成调色板，你可以传入:

```php
$panel
    ->colors([
        'primary' => '#6366f1',
    ])

$panel
    ->colors([
        'primary' => 'rgb(99, 102, 241)',
    ])
```

## 修改字体

默认情况下，我们使用 [Inter](https://fonts.google.com/specimen/Inter) 字体。你可以在[配置](configuration) 文件使用 `font()` 对其进行修改：

```php
use Filament\Panel;

public function panel(Panel $panel): Panel
{
    return $panel
        // ...
        ->font('Poppins');
}
```

所有 [Google Fonts](https://fonts.google.com) 都是可用的。

### 修改字体 Provider

[Bunny Fonts CDN](https://fonts.bunny.net) 用于提供字体服务。它符合 GDPR。如果你想使用 [Google Fonts CDN](https://fonts.google.com)，可以通过 `font` 方法的 `$proveder` 参数实现：

```php
use Filament\FontProviders\GoogleFontProvider;

$panel->font('Inter', provider: GoogleFontProvider::class)
```

如果你想在本地样式表中提供字体服务，可以使用 `LocalFontProvider`：

```php
use Filament\FontProviders\LocalFontProvider;

$panel->font(
    'Inter',
    url: asset('css/fonts.css'),
    provider: LocalFontProvider::class,
)
```

## 创建自定义主题

Filament 允许你通过编译自定义样式表替换默认的样式，修改 CSS 用以渲染 UI。自定义的样式表称为 "主题"。

主题使用 [Tailwind CSS](https://tailwindcss.com)、Tailwind Forms 插件、Tailwind Typography 插件和 [Autoprefixer](https://github.com/postcss/autoprefixer)。

要在面板中创建自定义主题，你可以使用 `php artisan make:filament-theme` 命令：

```bash
php artisan make:filament-theme
```

如果你有多个面板，你可以指定你要创建主题的面板：

```bash
php artisan make:filament-theme admin
```

该命令将在 `/resources/css/filament` 目录中生成 CSS 文件及 Tailwind 配置文件。你可以编辑这些文件自定义主题。它同时说明了如何在 Filament 中编译主题并注册。***、请遵照命令中的说明，完成安装过程**

## 非粘性顶部栏

默认情况下，顶部导航栏粘贴固定在页面顶部。你可以将下面样式使之可以滚动滑出视图之外：

```css
.fi-topbar {
    position: relative;
}
```

## 修改Logo

默认情况下，Filament 会在你使用你的应用名作为 Logo。

你也可以创建 `resources/views/vendor/filament-panels/components/logo.blade.phpp` 文件，用来自定义 logo:

```blade
<img src="{{ asset('/images/logo.svg') }}" alt="Logo" class="h-10">
```

## 禁用暗黑模式

要禁用暗黑模式切换，你可以使用[配置文件](configuration)：

```php
use Filament\Panel;

public function panel(Panel $panel): Panel
{
    return $panel
        // ...
        ->darkMode(false);
}
```

## 添加 favicon

要添加 favicon，你可以在[配置文件](configuration)中传入 favicon 的公共 URL：

```php
use Filament\Panel;

public function panel(Panel $panel): Panel
{
    return $panel
        // ...
        ->favicon(asset('images/favicon.png'));
}
```
