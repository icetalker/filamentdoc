## 概述

你可能想将多个 Entry 分组到一个 Fieldset 中。每个 Fieldset 默认都有一个标签、一条边框(border)以及 2 列宽的网格：

```php
use Filament\Infolists\Components\Fieldset;

Fieldset::make('Label')
    ->schema([
        // ...
    ])
```

## 在 Fieldset 中使用网格列宽

你可以使用 `columns()` 方法，在 Fieldset 中自定义 [Grid](grid):

```php
use Filament\Infolists\Components\Fieldset;

Fieldset::make('Label')
    ->schema([
        // ...
    ])
    ->columns(3)
```
