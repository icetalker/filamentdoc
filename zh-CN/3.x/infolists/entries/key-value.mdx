## 概述

Key-value Entry 允许你从一维 JSON 对象/PHP 数组中渲染键值对数据。

```php
use Filament\Infolists\Components\KeyValueEntry;

KeyValueEntry::make('meta')
```

<AutoScreenshot name="infolists/entries/key-value/simple" alt="Key-value entry" version="3.x" />

如果你在 Eloquent 中保存数据，你应该保证将 `array` [强制转换(cast)](https://laravel.com/docs/eloquent-mutators#array-and-json-casting)成模型属性：

```php
use Illuminate\Database\Eloquent\Model;

class Post extends Model
{
    protected $casts = [
        'meta' => 'array',
    ];

    // ...
}
```

## 自定义键(key)列标签

使用 `keyLabel()` 方法自定义键列的标签：

```php
use Filament\Infolists\Components\KeyValueEntry;

KeyValueEntry::make('meta')
    ->keyLabel('Property name')
```

## 自定义值(key)列标签

使用 `valueLabel()` 方法自定义值列的标签：

```php
use Filament\Infolists\Components\KeyValueEntry;

KeyValueEntry::make('meta')
    ->valueLabel('Property value')
```
