## 概述

头像组件用作渲染圆形图片或正方形图片，通常作为头像用来表示用户或者实体：

```blade
<x-filament::avatar
    src="https://filamentphp.com/dan.jpg"
    alt="Dan Harrin"
/>
```

## 设置头像圆角

头像通常默认为圆形，不过你也可以将 `circular` 属性设置为 `false` 使之成正方形：

```blade
<x-filament::avatar
    src="https://filamentphp.com/dan.jpg"
    alt="Dan Harrin"
    :circular="false"
/>
```

## 设置头像大小

默认情况下，头像大小为 "medium"。使用 `size` 属性你可以将其设置为 `sm`、`md` 或 `lg：

```blade
<x-filament::avatar
    src="https://filamentphp.com/dan.jpg"
    alt="Dan Harrin"
    size="lg"
/>
```

你也可以传入自定义的 size 类到 `size` 属性中：

```blade
<x-filament::avatar
    src="https://filamentphp.com/dan.jpg"
    alt="Dan Harrin"
    size="w-12 h-12"
/>
