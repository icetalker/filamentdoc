## 概述

Hidden 组件允许你在表单中创建 hidden 字段。

```php
use Filament\Forms\Components\Hidden;

Hidden::make('token')
```

请注意，如果用户使用浏览器开发者工具，该字段的值仍然是可编辑的。不建议使用该组件存储敏感信息或者只读信息。
